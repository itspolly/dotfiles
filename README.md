# dotfiles

My dotfiles.

## installing

### deps

* oh-my-zsh if you plan on using zsh
* git

### installing
**Please uninstall & backup your old files, including oh-my-zsh before running**

1. git clone https://github.com/nullpixel/dotfiles ~/dotfiles
2. chmod +x make.sh
3. ./make.sh

### updating

1. cd ~/dotfiles
2. git pull
3. :tada:
